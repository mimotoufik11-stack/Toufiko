# Contributing

Thank you for your interest in contributing to the Salafi Science Network website built with Astro. We appreciate your support and welcome contributions of all kinds. Please follow the guidelines below to help keep the project organized and effective.

## Getting Started

1. Fork the repository and clone it locally on your machine.
2. Install the project dependencies using `npm install`.
3. Create a new branch using a descriptive name for your changes. For example: `feature/add-new-navbar` or `fix/issue-description`.
4. Run `npm run dev` to start the development server and preview your changes locally.

## Contribution Guidelines

- Ensure your code follows the existing coding style and structure of the project.
- Verify that your changes work properly by testing the affected functionality.
- Keep commits clear and focused on specific features or fixes.
- Include appropriate documentation updates when adding new features or modifying existing ones.
- When possible, ensure your changes support both light and dark modes.

## Pull Request Process

1. Push your branch to GitHub.
2. Open a pull request (PR) describing your changes and link to any relevant issues.
3. Make sure your PR includes screenshots or screen recordings if your changes affect the UI.
4. Request a review from the project maintainers and address any feedback.
5. Merge when the PR is approved and CI checks pass.
